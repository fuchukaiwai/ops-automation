カテゴリはH1の10個とし、自動化したいタスクがあれば該当カテゴリ内に追記していく。

# 1) 監視・インシデント対応

* ★ **アラート一次対応の自動化**：CloudWatch Alarm → EventBridge → Lambda/SSM で切替・再起動・ログ採取まで自動実行。
* ★ **通知の集約と重複抑制**：同種アラートを5分でバッチ化、ノイズを減らす（Alarm → EventBridge → Step Functions）。
* **メトリクス異常検知（AI）**：履歴から“普段と違う”傾向を検知（Lookout for Metrics / 自前の時系列モデル）。
* **障害要約ボット（AI）**：CloudWatch Logs/アラート履歴を要約してDiscordに要旨＋推奨アクション投稿。
* **Runbook自動実行**：インシデント種別→標準手順を自動選択して実行（SSM Automation/Runbook）。

# 2) ログ管理・可観測性

* ★ **ログ集約＆ライフサイクル**：CloudWatch Logs → S3 へ定期Export、S3 Lifecycle/Glacierで保管コスト最適化。
* ★ **アプリ/OSログの定型チェック**：ログパターン（ERROR、Timeout）をCWL Insightsで日次サマリ化しSlack/Discord通知。
* **ログ要約（AI）**：長いログを“事象/影響/原因候補/次アクション”に自動要約。
* **自然言語→CWL Insightsクエリ（AI）**：“昨日の5xx多いURLは？”→クエリ生成・実行・結果可視化。
* **メトリクス自動可視化**：ダッシュボードをIaC化（CDK/CloudFormation）し、環境ごとに自動生成。

# 3) バックアップ／リストア

* ★ **バックアップの健全性点検**：Backup/AWS Backup/スナップショットの取得成否・世代数・暗号化有無を日次レポート。
* ★ **リストア演習の自動化**：テストVPCに自動復元→疎通テスト→結果を保存（Step Functions + Lambda）。
* **差分検証の要約（AI）**：本番 vs 復元環境の設定差分を説明文に要約して共有。
* **RPO/RTO 逸脱検知**：バックアップ間隔や復旧時間の逸脱を検知し自動チケット起票。

# 4) セキュリティ運用

* ★ **脆弱性レポート自動配信**：Inspector/SSM Patch Manager結果を集約→重要度順に要約・配信。
* ★ **IAMポリシー最小権限化**：Access Analyzerの提案取り込み→PR自動作成。
* **不審操作の要約（AI）**：CloudTrailから高リスク操作のシナリオ要約＆影響推定。
* **S3公開設定ウォッチ**：パブリックACL/ポリシー検知→自動修正 or 承認フローへ。

# 5) 構成・インベントリ／変更管理

* ★ **インベントリ自動収集**：AWS Config/SSM InventoryでOS/タグ/SG/AMI等を日次スナップショット化。
* ★ **構成ドリフト検出**：本番/ステージング差分をCSV出力→毎週サマリ。
* **タグ漏れ自動補完**：必須タグ監査→欠落時に自動付与 or Issue起票。
* **変更影響の要約（AI）**：差分から“影響範囲・リスク・試験観点”を文章化。

# 6) プロビジョニング／初期構築

* ★ **OSユーザー一括作成**：SSM Run Commandで初期ユーザー・sudo・鍵配布を自動化。
* ★ **ディレクトリ＆権限初期化**：業務標準のフォルダ構造/パーミッションを一括適用。
* **スクリプト配布基盤**：S3/CodeArtifact/SSM Docで運用スクリプトを配布・バージョン管理。
* **IaCレビュー支援（AI）**：CDK/CFnの差分から変更概要とリスクを自動レビューコメント。

# 7) 運用定常作業

* ★ **定例レポート自動生成**：CPU/メモリ/ディスク/コスト/チケット件数を週次PDFやMarkdownで生成。
* ★ **メンテナンス告知自動化**：EventBridgeスケジュール→関係者へテンプレ通知＋カレンダー登録。
* **Runbook → スクリプト化支援（AI）**：手順書からShell/Pythonの雛形を生成。
* **作業ログの自動整形（AI）**：SSM出力を要約して作業記録に整形。

# 8) コスト最適化

* ★ **未使用リソース検出**：孤立EIP/未アタッチEBS/Idle RDS等のリスト化と削除ワークフロー。
* ★ **スケジュール停止**：検証/開発環境の夜間停止・朝起動をアカウント横断で。
* **コスト異常の早期検知（AI）**：日次トレンドから外れ値を検知し要因候補を提示。
* **Savings Plans/RI 検討レポート**：購入提案を自動作成（Compute Optimizer/Cost Explorer）。

# 9) データベース運用

* ★ **DBヘルスチェック**：接続数/遅延/クエリTop-Nを日次レポート。
* **スロークエリ要約（AI）**：原因候補・インデックス案を文章化。
* **パラメータ逸脱監査**：本番のDBパラメータを基準化・逸脱検知。

# 10) ナレッジ・コミュニケーション

* ★ **自動議事録（AI）**：Discord/会議の要点を“決定事項/宿題/期限”で要約。
* ★ **チケット自動起票**：アラート→Jira/GitHub Issues をテンプレで作成。
* **Postmortem下書き（AI）**：時系列ログから障害振り返りの初稿を作成。
